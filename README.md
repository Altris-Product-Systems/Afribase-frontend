# Afribase Frontend

The official dashboard and documentation portal for Afribase Cloud Infrastructure. Built with Next.js, TypeScript, and high-performance aesthetics.

## Key Features

- **Project Dashboard**: Manage your databases, auth metrics, and storage.
- **Interactive Documentation**: Comprehensive guides for all Afribase services.
- **Multi-SDK Support**: Implementation guides for JavaScript, Python, and Dart.
- **Developer First**: Optimized for rapid cloud application development.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: Vanilla CSS with custom design tokens
- **Icons**: [Lucide React](https://lucide.dev)
- **Deployment**: Vercel
